## вҡЎ Installation Guide

<div align="center">
  <img src="https://capsule-render.vercel.app/api?type=rect&color=42A5F5&height=2&section=header" width="80%" />
</div>

### рҹ“Ӣ Prerequisites

Before installing, ensure you have the following installed on your system:

| Requirement | Version | Download |
|-------------|---------|----------|
| Python | 3.8+ | [python.org](https://python.org) |
| pip | Latest | Included with Python |
| Git | Latest | [git-scm.com](https://git-scm.com) |

### рҹҡҖ Quick Start

#### Step 1: Clone the Repository

```bash
git clone https://github.com/issu321/Billing-App-Enterprise-Level.git
cd Billing-App-Enterprise-Level
```

#### Step 2: Create Virtual Environment (Recommended)

```bash
# On Windows
python -m venv venv
venv\Scripts\activate

# On macOS / Linux
python3 -m venv venv
source venv/bin/activate
```

#### Step 3: Install Dependencies

```bash
pip install -r requirements.txt
```

#### Step 4: Configure Environment

```bash
cp .env.example .env
# Edit .env with your configuration
```

#### Step 5: Initialize Database

```bash
python run.py --init-db
```

#### Step 6: Run the Application

```bash
python run.py
```

#### Step 7: Access the Application

Open your browser and navigate to:

```
http://localhost:5000
```

Default credentials:
- **Username:** `admin`
- **Password:** `admin123`

> вҡ пёҸ **Security Note:** Change default credentials immediately after first login.

### рҹҗі Docker Deployment (Optional)

```bash
# Build Docker image
docker build -t billing-app-enterprise .

# Run container
docker run -p 5000:5000 billing-app-enterprise
```

### вҳҒпёҸ Production Deployment

For production environments, refer to the [DEPLOYMENT.md](docs/DEPLOYMENT.md) guide for:
- рҹ–ҘпёҸ **Gunicorn** WSGI server configuration
- рҹ”„ **Nginx** reverse proxy setup
- рҹ”җ **SSL/TLS** certificate installation
- рҹ—„пёҸ **Database** optimization
